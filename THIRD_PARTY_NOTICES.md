## Reference repositories

The optional `RepoInspiredEnhancements` scene layer is original project code,
but its organization and visual approach were informed by public open-source
Three.js/R3F projects:

- `hyeonsangjeon/Repolis` for city-block composition ideas.
- `dgreenheck/simcity-threejs-clone` for tile-based street and city-detail organization.
- `dgreenheck/ez-tree` for procedural vegetation structure and wind-sway concepts.
- `Yashparmar1125/Car-Game-ThreeJS` for vehicle motion cue ideas.
- `pmndrs/ecctrl` for character-controller-informed pedestrian proportions.
- `zappar-xr/react-three-example-image-tracking-business-card` for spatial card placement ideas.
- `pmndrs/drei` for `<Html />` and `<Billboard />` style spatial UI patterns.

No source files or binary assets from these repositories were copied into this
project in this pass.
